## 2.select sort
```
bool selectSort(vector<int> &array){
    int n = array.size();

    for(int i = n-1;i >= 1; --i){
        for(int j = 0;j < i; ++j){
            if(array[j] > array[i]){
                swap(array[i],array[j]);
            }
        }
    }

    return true;
}
```

bool insertSort(vector<int> &array){
